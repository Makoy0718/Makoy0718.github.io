HTML Basics

HTML stands for HyperText Markup Language.

HTML is the markup language that we use to structure and give meaning to our web content, for example defining paragraphs, headings, and data tables, or embedding images and videos in the page.

HTML elements are represented by tags.

Example of a basic HTML structure:

<!DOCTYPE html> <html></html>

More HTML tags include

  • <head>
  • <body>
  • <p>
  • <a>
  • <div>
CSS Fundamentals

CSS stands for Cascading Style Sheets

CSS is a language of style rules that we use to apply styling to our HTML content, for example setting background colors and fonts, and laying out our content in multiple columns.

CSS can control the layout of multiple web pages all at once.

Example of CSS syntax:

el-selector { propiedad-ejemplo: valor; }

Common CSS properties include:

  • color
  • font-size
  • margin
  • padding
  • border
JavaScript Intro

JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.

It is an essential part of web applications.

JavaScript can update and change both HTML and CSS.

Example of a JavaScript function:

funcion mi_funcion() { alert("Hello World!"); }

JavaScript is often used for:

  • Client-side scripting
  • Server-side programming
  • Mobile app development
  • Game development
  • Data visualization
Responsive Design

It allows web pages to render well on a variety of devices and window or screen sizes.

Media queries are used to apply different styles for different devices.

Example media query:

@media (max-width: 600px) { body { background-color: lightblue; } }

Key components of responsive design include:

  • Fluid grids
  • Flexible images
  • Media queries
APIs

API stands for Application Programming Interface.

An API allows different software programs to communicate with each other.

Web APIs provide access to web services.

Example of a simple API call:

fetch('https://api.example.com/data')

Common API methods include:

  • GET
  • POST
  • PUT
  • DELETE
  • PATCH
